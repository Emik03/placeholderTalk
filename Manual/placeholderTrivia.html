On the Subject of Placeholder Talk

Maybe we should've waited for a Stable release...

The module has a display containing a message alongside 4 buttons below it. The message consists of two unique phrases, the first one describes an ordinal and can be solved within this first page. The second phrase containing a message found in pages 2-6. Your task is to identify which button (1-4) should be pushed. Figure out the two variables needed then apply the formula: Push button = (FIRST PHRASE + SECOND PHRASE) mod 4. Press fourth if you get 0.

Deducing the module's first phrase

To figure out the value from the first phrase, do the following steps:

  1. Start with N=1
  2. Add 1 to N for every strike.
  3. Multiply N by the number of batteries.
  4. Add 1 to N for every "\" only if the second phrase has an odd number of "/".
  5. If the second phrase is 3 lines long, follow the instructions within it.
  6. Do (N - M) with whichever condition is fulfilled:
Words within first phrase M is equal to
"" -1
"IS IN THE" 0
"IS ON THE"
"IS IN UHHH"
"IS" 1
"IS AT THE"
"IS INNNN"
"IS THE IN" 2
"IN IS"
"IS IN..."
"IS IN" 3
"THE"
"IS FIRST- NO I MEAN"
"IN" 27
"UHHH IS IN" 30
